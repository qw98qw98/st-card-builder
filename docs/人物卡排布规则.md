 排布逻辑（从上到下）

 按文章描述的 SillyTavern Prompt 排布规则：

 ┌─ 世界书常数条目（strategy=constant，按 Order 排序）─── 可收起
 ├─ 用户描述 / 人设（如存在）
 ├─ 角色定义之前的世界书（position=0，按 Order 排序）─── 可收起
 ├─ 角色描述（charDesc）
 ├─ 角色定义之后的世界书（position=1，按 Order 排序）─── 可收起
 ├─ 性格（personality）
 ├─ 场景（scenario）
 ├─ 对话示例（mes_example）
 ├─ 开场白（first_mes）
 ├─  ··· 聊天记录占位 ···  （虚线分隔区域）
 ├─ 作者注释之前的世界书（position=5，按 Order 排序）─── 可收起
 ├─ 创作者备注 / 作者注释（creatorNotes）
 ├─ 作者注释之后的世界书（position=6，按 Order 排序）─── 可收起
 ├─ @D 深度插入的世界书（position=4，按 depth 排序）─── 可收起
 └─

 Position 映射（来自已有代码）：
 - 0: 角色定义之前
 - 1: 角色定义之后
 - 2: 前 EM（较少使用）
 - 3: 后 EM（较少使用）
 - 4: @D 深度插入
 - 5: 前作者注释
 - 6: 后作者注释

 世界书条目可视化

 每个世界书条目显示：
 - 条目标题（comment）
 - Order 数值（醒目标签）
 - 触发关键词（keys）
 - 来源标签（source: manual/ai/import，复用已有 pill 样式）
 - 策略标签（selective/constant/vectorized）
 - 优先级（prob，数值越高越优先保留）
 - 内容预览（content 截断显示）
 - 点击可收起/展开详细内容

 同一 Position 内按 Order 数值从小到大排列（Order 小的在上，Order 大的在下、影响更强）。

 非世界书区域（常驻显示）

 角色描述、性格、场景、对话示例、开场白、创作者备注等，直接显示内容摘要或字符数，不可收起。空字段显示灰色占位提示。
